About me

I'm Malte, a 19 year old IT student and hobbyist open source developer. All my open source projects can be found on my github profile.

Below, you can find a list of all my repositories, and a description of them.

aur-packages

aur-packages contains all PKGBUILDs and extra files for my packages in the ArchLinux User Repository.

dotfiles

The dotfiles repository contains some of the configuration files of my main workstation.

Feel free to build your wn configuration on top of my work, since the idea behind this repository is to share them so others (and maybe you) can benefit from it.

flyingpaste

flyingpaste is a nopaste service with syntax highlighting support.

minecraft_server_status

minecraft_server_status is a PHP function which returns basic information about a Minecraft SMP server.

OSDetect

OSDetect is a python class to get some information about the operating system currently running.

rahmenwerk

rahmenwerk is a PHP framework which is the base of all my web applications. You can find it's website here: malte70.github.io/rahmenwerk

simple-chat

simple-chat is a simple, python-powered single-room Chat software. It was written after I visited the IT class of some friends of mine where they started with network programming. They used Java, But I prefered Python, and in my opinion, the code they wrote is much less readable.

TuxInSpace

TuxInSpace is my first try in game programming, a space shooter like the classic space invaders.

WindowsInfo

WindowsInfo displays some information about a windows system, including the registered owner and the license key.
